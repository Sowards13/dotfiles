## Homefiles 

## Files included 
 - .Bash_aliases: File that contains shortcuts for navigating your directory and clearing your terminal 
 - install_bashaliases.sh: Bash script that will automate the process of symbolically linking the repo's alias file to the user's home directory. 

## Functionality 
To clear your terminal, some of the aliases are: 
- c 
- cl  
- clr 
- ckear 

To navigate your directory, some of the aliases are: 
- ..='cd ..'
- ...='cd ../..'
- ....='cd ../../../'
