# Dotfiles  
- A collection of installation files and configuration scripts to customize a users environment.

## RepositoryContents 
- This repository is organized into specific folders to keep confgurations easy to manage. This repo primarily uses shell and and vim script. 

- homefiles/: Contains user specific configuration files that are intended to be copied to the home directory (Environment aliases). 

- sys-installs/: Conatins system level instal scripts. 

- vim-config/: Dedicated customizations for the Vim text editor. 

- setup.sh/: Main executable script that is used to symlink the specific dotfiles and manage the installation procedure.
## How to clone and utilize 
1. clone the repo 
- git clone https://github.com/Sowards13/dotfiles 
- cd /dotfiles
